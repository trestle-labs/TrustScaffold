# UAT Bundle Run Sheets

This folder contains compact, single-document run sheets for the recommended UAT bundles.

Each run sheet combines:

- the organization maturity profile
- the environment profile
- the exact answer path
- the key warnings and outcomes to verify

Use these when you want a tester to run one complete scenario without switching between multiple files.

Track overall execution state in [../UAT_TRACKER.md](../UAT_TRACKER.md).
